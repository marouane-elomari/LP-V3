# 🗺️ Sitemap Dynamique - Hostinger

## ⚡ Démarrage Rapide

### 1️⃣ Vérifier la Configuration
```bash
chmod +x check-hostinger-setup.sh
./check-hostinger-setup.sh
```

### 2️⃣ Configuration Initiale (Première fois uniquement)
```bash
# Configurer le remote git
cd dist/
git init
git remote add hostinger <VOTRE_URL_GIT_HOSTINGER>
cd ..
```

### 3️⃣ Déployer
```bash
npm run deploy:hostinger
```
